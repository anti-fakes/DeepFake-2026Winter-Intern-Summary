# Corvi+ 모델 추가분석

## 배경 및 목적
사용한 탐지모델 중 가장 뛰어난 성능을 보이는 Corvi+에 대한 추가적인 분석 진행
학습된 가중치나 활성화된 채널의 탐지에 대한 실제 기여도를 확인함으로써 모델의 한계점이나 향후 개선 방향 제시

---

## 1. Corvi+ 모델의 학습된 가중치 확인

- FC layer의 2048개 채널 중 Fake 방향(양수 weight) 채널은 약 35개에 불과
- 상위 소수 채널이 전체 기여도의 상당 부분을 차지하는 것을 확인
- 모델은 Fake 판별을 위해 소수의 강한 증거 채널에 의존하는 구조를 가짐
- 아래는 학습이 완료된 후 FC layer의 가중치 분포를 히스토그램으로 나타낸 결과

## 2. Channel-wise Attribution Analysis

### 실험 개요
- 데이터셋 전체에 대해 TP, TN, FP, FN 4개의 그룹으로 분리
- 각 그룹의 이미지별로 활성화된 채널에 대한 Attribution (activation × weight) 계산
- 그룹의 채널별 Attribution 평균값을 계산, 상위 채널 확인
- FP/FN(오분류)에서 튀는 채널이 TP/TN(정분류)과 어떻게 다른지 경향성 확인

### 사용 데이터셋
- **TP / FN**: GPT, Z-image
- **FP / TN**: COCO, Redcaps, Synthbuster

## 6. 종합 결과

- TP, TN, FP, FN 4분류 모두에서 Attribution 상위 10개 채널의 합이 전체 Attribution의 50% 이상 차지한다.
- Attribution 하위 10개 채널의 합은 전체의 6% 이하를 차지한다.
- 채널별 Activation 값은 대부분의 채널에서 TP > FP > FN > TN 경향을 보인다.
- GPT, Z-image의 FN 이미지들은 20개 이상의 채널에서 0.1 이하의 값을 가진다.
  - 학습한 Fake 아티팩트가 존재하지 않는 것처럼 보인다.
- Fake 이미지(GPT, Z-image)에 대해 사용되는 상위 채널이 유사하다.
  - CH1192, CH7, CH604 등 상위 공통채널은 거의 동일하다.
